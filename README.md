Spice Palace is a premium, highly interactive digital storefront designed for an authentic Indian fine-dining restaurant.

Rather than building a traditional website where users must endure loading screens for every click, I engineered this project as a Single-Page Application (SPA). This modern architecture ensures that browsing the menu, exploring the restaurant's details, and booking a table happen instantly on a single, fluid page, mimicking the seamless experience of a native mobile app.

Core Highlights:

Dynamic Interactivity: I implemented a custom JavaScript tab system that allows users to instantly filter the food menu by category (Starters, Mains, Desserts) without ever reloading the browser.

Intelligent Animations: To make the site feel alive without sacrificing performance, I utilized the modern IntersectionObserver API. This ensures that text and images only animate onto the screen when the user scrolls down to them, saving system resources.

Smart Booking System: The reservation system isn't just visual; it includes client-side JavaScript validation that prevents users from selecting past dates and provides a slick, animated success notification upon completion.

The Development Philosophy:
To demonstrate a rigorous understanding of core web fundamentals, I built this entire application 100% from scratch. I deliberately avoided using heavy JavaScript frameworks (like React) or pre-built CSS libraries (like Bootstrap).

Every line of semantic HTML5, every responsive CSS3 Grid layout, and all the Vanilla JavaScript logic was written by hand. The result is a lightweight, lightning-fast application that looks stunning on a 4K desktop monitor and functions flawlessly on a mobile phone.
